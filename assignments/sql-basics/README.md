# 📘 Assignment: SQL Basics & Database Queries

## 🎯 Objective

Learn the fundamentals of SQL (Structured Query Language) by writing queries to retrieve, filter, and manipulate data. You'll work with a student database to practice SELECT, WHERE, INSERT, UPDATE, and DELETE statements—essential skills for managing data in real-world applications.

## 📝 Tasks

### 🛠️ Write SELECT Queries and Filter Data

#### Description
Write SQL SELECT statements to retrieve specific data from the database. Practice filtering records with WHERE clauses, sorting results, and using aggregate functions to analyze data.

#### Requirements
Completed queries should:

- Retrieve all students or filtered results using SELECT and WHERE
- Sort results using ORDER BY (ascending/descending)
- Use aggregate functions (COUNT, SUM, AVG, MAX, MIN) to analyze grades
- Filter records using comparison operators (=, <, >, <=, >=, !=)
- Combine multiple conditions using AND/OR operators

### 🛠️ Modify Data with INSERT, UPDATE, and DELETE

#### Description
Write SQL statements to create new records, modify existing data, and remove records from the database. Understand the importance of careful updates and deletions.

#### Requirements
Completed implementation should:

- INSERT new student records with all required fields
- UPDATE existing student grades or information
- DELETE records based on specific conditions
- Demonstrate understanding of potential data loss from DELETE operations
- Write queries that are safe and specific (use WHERE clauses appropriately)
